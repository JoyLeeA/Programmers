1. 없어진 기록 찾기

```mysql
SELECT O.ANIMAL_ID, O.NAME FROM ANIMAL_INS AS I RIGHT JOIN ANIMAL_OUTS AS O ON I.ANIMAL_ID = O.ANIMAL_ID WHERE I.DATETIME IS NULL
```



2. 있었는데요 없었습니다

```mysql
SELECT I.ANIMAL_ID, I.NAME FROM ANIMAL_INS AS I JOIN ANIMAL_OUTS AS O ON I.ANIMAL_ID = O.ANIMAL_ID WHERE I.DATETIME > O.DATETIME ORDER BY I.DATETIME ASC
```



3. 오랜 기간 보호한 동물(1)

```mysql
SELECT I.NAME, I.DATETIME FROM ANIMAL_INS AS I LEFT JOIN ANIMAL_OUTS AS O ON I.ANIMAL_ID = O.ANIMAL_ID WHERE O.DATETIME IS NULL ORDER BY I.DATETIME ASC LIMIT 3
```



4. 보호소에서 중성화한 동물

```mysql
SELECT I.ANIMAL_ID, I.ANIMAL_TYPE, I.NAME FROM ANIMAL_INS AS I JOIN ANIMAL_OUTS AS O ON I.ANIMAL_ID = O.ANIMAL_ID WHERE I.SEX_UPON_INTAKE LIKE "Intact%" AND (O.SEX_UPON_OUTCOME LIKE "%Spayed%" OR O.SEX_UPON_OUTCOME LIKE "%Neutered%") ORDER BY I.ANIMAL_ID ASC

SELECT A.ANIMAL_ID, A.ANIMAL_TYPE, A.NAME FROM ANIMAL_INS A , ANIMAL_OUTS B WHERE A.ANIMAL_ID = B.ANIMAL_ID AND A.SEX_UPON_INTAKE LIKE 'I%' AND (B.SEX_UPON_OUTCOME LIKE 'N%' OR  B.SEX_UPON_OUTCOME LIKE 'S%')
```
